# 프로젝트 개요

## 팀

- **팀명: Aice**
- 해커톤 출품 프로젝트의 발표용 웹사이트를 제작한다.

## 발표 형식

- 자유 인터랙티브 웹 (PPT식 슬라이드 덱 아님)
- 슬라이드 단위로 구성하되, 슬라이드 간 전환도 애니메이션으로 연출

## 기술 스택

- 순수 HTML / CSS / JavaScript + GSAP
- 빌드 없음, 오프라인 동작 필수 (자세한 내용은 `AGENTS.md`)

## 핵심 컨셉

- 팀명 **Aice** 안에 **AI**가 들어있다.
- **"Ace(에이스)" + "i" = "Aice"** — 에이스에 AI를 더한 팀.
- 이 워드플레이를 인트로(슬라이드 1) 애니메이션의 핵심 모티프로 사용한다.

## 발표 주제 · 프로젝트

- **주제: 에너지 절약.**
- **프로젝트: 에너지 사용량 모니터링·관리 앱** — 전력 사용을 추적하고 관리해주는 앱/대시보드.
- 핵심 메시지: 에너지 절약에는 수많은 요소가 필요하지만, 결국 가장 중요한 것은 **관리**다.
  이 인사이트를 슬라이드 2에서 시각화한다(흩어진 요소 → 하나로 응축 → "관리" 강조).

## 발표 흐름 (현재까지)

- 슬라이드 1 — 인트로(팀 Aice)
- 슬라이드 2 — 핵심 인사이트 "관리"
- 슬라이드 3 — "측정" (피터 드러커 인용: 측정 → 관리 → 개선). 슬라이드 2와 제품 소개를 잇는 다리.
- 슬라이드 4 이후 — 솔루션(제품 소개) → 데모 → 차별점 → 마무리 (미정)

## 미정 사항

- 슬라이드 4(솔루션·제품 소개) 이후 구성
